# Blog Content

This directory contains all blog posts for the portfolio site.

## Adding a New Blog Post

1. Create a new `.md` file in this directory
2. Use kebab-case for the filename (e.g., `my-awesome-post.md`)
3. Add frontmatter at the top of the file:

```markdown
---
title: "Your Post Title"
date: "YYYY-MM-DD"
description: "A brief description of your post"
tags: ["tag1", "tag2", "tag3"]
readTime: "X min read"
---

# Your content starts here

Write your blog post content in markdown...
```

## Frontmatter Fields

- **title** (required): The title of your blog post
- **date** (required): Publication date in `YYYY-MM-DD` format
- **description** (required): A brief summary/description of the post
- **tags** (optional): Array of tags for categorizing the post
- **readTime** (optional): Estimated reading time (e.g., "5 min read")

## Markdown Features

You can use all standard markdown features:

- **Bold** and *italic* text
- [Links](https://example.com)
- Code blocks with syntax highlighting
- Images
- Lists (ordered and unordered)
- Blockquotes
- Tables
- And more!
